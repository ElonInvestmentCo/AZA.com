The PAYVORA logo is currently rendering far too small in the header on both the website and the mobile app. It is difficult to see and weakens the brand identity.

## TASK

Increase the size and prominence of the official PAYVORA logo everywhere it appears without changing the logo design itself.

### Requirements

1. Audit every place where the PAYVORA logo is used.

This includes:
- Website navbar
- Mobile website header
- Native mobile app header
- Landing page
- Authentication screens
- Dashboard
- Sidebar
- Drawer menu
- Settings
- Splash screen
- Loading screens
- Any reusable Logo component

2. Increase the visual size of the logo.

The logo should:
- Be bold and immediately recognizable.
- Occupy an appropriate amount of header space.
- Be approximately 2–3× larger than its current display where it is too small.
- Maintain the original SVG aspect ratio.
- Never stretch or distort.
- Never become blurry or pixelated.
- Always render from the SVG asset (avoid rasterized versions where possible).

3. Update the shared Logo component.

If multiple logo implementations exist, consolidate them into a single reusable Logo component so every screen uses the same sizing rules.

4. Responsive sizing.

Use responsive sizing for:
- Mobile phones
- Tablets
- Desktop
- Large desktop displays

The logo should scale appropriately instead of remaining tiny.

5. Header layout.

Adjust the navbar/header layout if necessary so the larger logo:
- Has adequate padding
- Is vertically centered
- Is horizontally aligned
- Does not overlap navigation items
- Does not push the hamburger menu or navigation off-screen

6. Light & Dark Mode.

Ensure the logo remains highly visible on both light and dark backgrounds.

7. Splash Screen.

Use the same official SVG logo for the native splash screen and increase its size so it is visually prominent while maintaining proper margins and aspect ratio.

8. Verification.

After completing the changes:
- Test on iPhone and Android screen sizes.
- Test the responsive website on mobile, tablet, and desktop.
- Verify there is no clipping, overflow, distortion, or layout shift.

## Important

- Do NOT redesign or edit the PAYVORA logo.
- Do NOT change the logo artwork.
- Do NOT change the colors.
- Only improve its implementation, display size, responsiveness, and visibility.

## Deliverables

Provide:
- Every file modified
- Components updated
- New responsive sizing values
- Before/after dimensions for the logo
- Confirmation that the logo now displays prominently across the website, mobile web, native mobile app, and splash screen.
