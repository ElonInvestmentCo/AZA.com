Implement a premium fintech dark mode across the entire AZA app using the device's system appearance settings only.

IMPORTANT REQUIREMENTS:

1. DO NOT add any Dark Mode toggle, switch, button, menu item, or settings page anywhere in the app.

2. Automatically detect the operating system theme:
   - iOS: Follow the device Appearance setting (Light/Dark).
   - Android: Follow the device system theme.
   - Web/Desktop: Follow prefers-color-scheme.
   - React Native/Expo: Use Appearance API or useColorScheme().
   - Flutter: Use ThemeMode.system.
   - Next.js/Web: Use matchMedia('(prefers-color-scheme: dark)').

3. The app must instantly and smoothly switch themes whenever the user changes their device appearance settings without restarting the app.

4. Theme changes should animate smoothly:
   - 250ms–350ms transition duration.
   - Fade colors smoothly.
   - No screen flashing.
   - No layout shifts.
   - Preserve current screen state.

5. Create a premium fintech dark theme inspired by modern trading apps such as Stripe, Revolut, Coinbase, Robinhood, and Wealthfront.
