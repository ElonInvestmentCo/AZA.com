Implement a fully native iOS Dynamic Island and Live Activities experience for all supported iPhone models (including all current and future devices that support Dynamic Island), following Apple’s Human Interface Guidelines and ActivityKit best practices.

General Requirements

* Support all Dynamic Island presentation modes:
    * Minimal
    * Compact
    * Expanded
    * Lock Screen Live Activity
* Automatically adapt layouts, spacing, typography, animations, and icon sizes for every supported iPhone model and Dynamic Island size.
* Ensure all UI elements are responsive and scale perfectly across different screen sizes and Dynamic Island dimensions without clipping, truncation, overlap, or misalignment.
* Respect Safe Areas, Dynamic Type, accessibility settings, and different display zoom modes.
* Support both Light Mode and Dark Mode using PayVora’s premium fintech design language.
* Use smooth native iOS animations, haptic feedback, and SF Symbols where appropriate.
* Ensure Live Activities continue updating correctly while the app is in the background or when the device is locked.
* Display only active, relevant events and automatically dismiss completed activities after an appropriate delay.
* If multiple Live Activities exist simultaneously, intelligently prioritize and manage them according to Apple’s system behavior.
