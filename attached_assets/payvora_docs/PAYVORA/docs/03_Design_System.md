# Design System

## Color Palette

PAYVORA uses a clean, high-contrast color system optimized for trust, readability, and financial clarity.

### Primary Colors
- Primary Blue: #2F6BFF
- Deep Navy: #0B1B3A
- White: #FFFFFF
- Black: #000000

### Semantic Colors
- Success: #18C37E
- Warning: #FFB020
- Error: #FF4D4F
- Info: #2DD4FF

### Neutral Scale
- Gray 900: #111827
- Gray 700: #374151
- Gray 500: #6B7280
- Gray 300: #D1D5DB
- Gray 100: #F3F4F6

---

## Typography

- Font Family: Inter / SF Pro Display
- Monospace: JetBrains Mono

### Scale
- H1: 28px
- H2: 24px
- H3: 20px
- Body: 16px
- Small: 14px
- Caption: 12px

### Rules
- Prioritize readability
- Avoid decorative typography
- Maintain consistent hierarchy

---

## Iconography

- Line-based icons only
- 1.5–2px stroke width
- Rounded endpoints
- Must scale to 16px clarity
- No filled icons except status indicators

---

## Spacing

- Base unit: 4px
- Scale: 4, 8, 12, 16, 24, 32, 48, 64
- Consistent vertical rhythm required

---

## Grid System

- Mobile: 4 columns
- Tablet: 8 columns
- Desktop: 12 columns
- 16px outer margins

---

## Components

- Buttons
- Cards
- Inputs
- Tables
- Modals
- Navigation bars
- Transaction elements

---

## Buttons

- Primary, Secondary, Ghost, Destructive
- Minimum height: 44px
- Clear states: hover, active, disabled, loading

---

## Cards

- Financial data containers
- Subtle elevation
- 12–16px radius
- Structured hierarchy

---

## Inputs

- Floating labels
- Real-time validation
- Secure input modes
- Auto-format financial fields

---

## Navigation

- Bottom navigation (mobile)
- Sidebar (desktop)
- Max 5 primary tabs
- Wallet always accessible

---

## Elevation

- Layer system:
  - Level 0: background
  - Level 1: cards
  - Level 2: modals
  - Level 3: overlays

---

## Shadows

- Soft, minimal shadows only
- No heavy blur effects
- Example:
  - 0 2px 8px rgba(0,0,0,0.08)
  - 0 4px 16px rgba(0,0,0,0.12)

---

## Border Radius

- Small: 8px
- Medium: 12px
- Large: 16px
- Full: 999px

---

## Dark Theme

- Primary mode
- Deep navy / black backgrounds
- White text
- Blue accents
- High contrast for readability

---

## Light Theme

- White background
- Dark text
- Subtle gray surfaces
- Minimal shadows

---

## Accessibility

- WCAG 2.1 AA compliance
- 4.5:1 contrast minimum
- 44px touch targets
- Screen reader support
- No color-only indicators

---

## Motion Principles

- 150–300ms transitions
- Ease-out easing
- Purposeful motion only
- No decorative animations
