BOTTOM SHEET / ACCOUNT POPUP
====================================================

Reduce the height of the Account popup.

It should initially open around 60–70% of the screen height instead of nearly full screen.

Implement a premium native bottom sheet experience.

Requirements:

• Drag handle at the top
• Users can drag the sheet downward
• The sheet should follow the user's finger smoothly
• Rubber-band effect near limits
• Velocity-based dismissal
• Snap points:
    - 35%
    - 65%
    - 95%

Dismiss when dragged below the lowest snap point.

Support:

• Backdrop blur
• Fade animation
• Spring animation
• Native gesture handling
• No lag
• 60 FPS animations
• Haptic feedback (where supported)

Scrolling inside the sheet should not conflict with dragging.
